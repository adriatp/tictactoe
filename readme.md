# TicTacToe

## Versions

It uses `node v18.13.0`. Can be applied with `nvm use`.

## Linter

* Installation

```bash
npm init @eslint/config -- --config semistandard
```

* Run eslint all files

```bash
npx eslint .
```

* Quick fix all files

```bash
npx eslint . --fix
```
